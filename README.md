## 📌 Project Overview

AI-Powered Chest X-Ray CDSS is a Clinical Decision Support System developed to help detect diseases such as pneumonia from chest X-ray images using Artificial Intelligence. The system allows users to upload chest X-ray images, analyze them using a CNN-based deep learning model, and generate prediction results with confidence scores and heatmap visualization.

The project combines medical image analysis with patient clinical data to support faster and more accurate diagnosis. It also provides treatment suggestions, severity classification, secure authentication, and downloadable reports through a modern full-stack web application.

### AI Analysis Module

---

## 📊 Workflow

1. User uploads a chest X-ray image
2. Image is processed and analyzed using CNN model
3. Prediction results are generated
4. Heatmap highlights affected areas
5. Patient vitals are combined with prediction results
6. Final diagnosis and treatment suggestions are displayed
